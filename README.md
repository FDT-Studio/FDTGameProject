## 关于二次开发
- “二次开发”指对**游戏内容**进行更改后打包分发的行为，如修改了剧情/替换了CG等。
- **本游戏禁止进行二次开发**
- 如果您想对本游戏的仓库做出贡献，欢迎提交Pull Request。
- 如果您想参与后续的游戏创作，欢迎联系我们。
